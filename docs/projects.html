Projects

Current and past projects in which I was involved:
  • 2022 - Clog

    • Clog is a fully declarative static code checking framework for C. It combines Datalog reasoning with syntactic pattern matching in Clang to enable the definition of bug detectors for C, in a manner that is independent of any representation of programs, other than source code.

  • 2020 - JavaDL

    • JavaDL is a programming language and a runtime for defining static bug checkers for Java. It is an extension of Datalog with syntactic patterns and semantic predicates. The runtime supports alternative evaluation modes: incremental and exhaustive.

  • 2019 - MetaDL

    • MetaDL is an extension of the Datalog language with syntactic patterns, to enable the analysis of Datalog programs inside Datalog. We are currently exploring ways to use the pattern syntax to analyze other programming languages and to express code transformations.
