I am a compiler engineer with a PhD in static program analysis.

I did my PhD at Lund University, under the supervision of Christoph Reichenbach and Emma SÃ¶derberg. In my thesis, I explore declarative techniques for building static code checkers that have comparable performance and analysis quality with state-of-the-practice tools. The aim of this work is to reduce the overhead of defining bug detectors, and thus enable more software developers to customize static code checkers for the needs of their projects.

Before this, I developed compilers for Arm (2012-2018) and Movidius (2010-2012). I contributed to all stages of a compiler pipeline, from machine code generation for highly specialized architectures (GPUs and VPUs) to graphics-specific language frontends and optimizations.
